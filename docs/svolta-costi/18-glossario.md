# Capítulo 18 — Glossário

## 18.1 Termos do Sistema
Os termos abaixo descrevem conceitos de negócio utilizados no domínio.

| Termo | Definição |
|-------|-----------|
| **Data de Referência** | Data contábil aplicada ao lote importado. |
| **Origem** | Primeiro nível de classificação do produto. |
| **Família** | Segundo nível de classificação do produto. |
| **Agrupamento** | Terceiro nível de segmentação analítica. |
| **Alerta de Alta** | Item com variação percentual acima de 5%. |

## 18.2 Termos Técnicos
Os termos abaixo descrevem elementos técnicos da implementação.

| Termo | Definição |
|-------|-----------|
| **Supabase** | Plataforma que fornece Auth, PostgREST e Postgres gerenciado. |
| **Upsert** | Operação de inserir ou atualizar conforme conflito de chave. |
| **RLS** | Row-Level Security aplicada por política no Postgres. |
| **SPA** | Single-Page Application executada no navegador. |
| **Schema Cache** | Cache de metadados usado pelo PostgREST para relações. |
